## Overview
This supplementary material involves some prediction results produced by DBR-TAD on three datasets: Thumos14, ActivityNet-1.3, and EPIC-Kitchens 100. This document outlines the folder structure.

## Folder Structure
- ActivityNet-1.3/
  - contains predictions on the ActivityNet-1.3 dataset using features extracted by TSP. Annotation information for the ActivityNet-1.3 dataset can be obtained from http://ec2-52-25-205-214.us-west-2.compute.amazonaws.com/files/activity_net.v1-3.min.json

- EPIC-Kitchens_100/Noun/
  - contains predictions for the noun task on the EPIC-Kitchens 100 dataset using features extracted by SlowFast. Annotation information for the EPIC-Kitchens 100 dataset can be obtained from https://github.com/epic-kitchens/epic-kitchens-100-annotations

- EPIC-Kitchens_100/Verb/
  - contains predictions for the verb task on the EPIC-Kitchens 100 dataset using features extracted by SlowFast. Annotation information for the EPIC-Kitchens 100 dataset can be obtained from https://github.com/epic-kitchens/epic-kitchens-100-annotations

- Thumos14/
  - contains predictions on the Thumos14 dataset using features extracted by VideoMAEv2. Annotation information for the Thumos14 dataset can be obtained from https://www.crcv.ucf.edu/THUMOS14/download.html
