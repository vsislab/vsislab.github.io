From Semantically-Reasonable Plan to Physically-Valid Manipulation: A Diffusion-Based Framework for LLM-Assisted Object Rearrangement

Chenkun Zhao1, Shuo Yang1, Dayou Li1, Ran Song1, Xiaolei Li1*, Penkun Wei1, Wei Zhang1 1School of Control Science and Engineering, Shandong University  

Abstract

In many scenarios, a robot is asked to rearrange objects into human-preferred configurations while such objects and configurations may be previously unseen. Most existing methods for object rearrangement only predict the goal states of objects through supervised learning, while not considering the order and trajectories of object manipulation. Consequently, they can only handle simple scenarios. To address this issue, we propose DiffLOR, a diffusion-based framework for LLM-assisted object rearrangement that leverages diffusion models and LLMs not only to predict the goal states of objects, but also to generate trajectory sequences for robotic manipulation. Specifically, we propose an automatic data generation pipeline which uses an LLM to generate rearrangement configurations and design a transformer-based diffusor to predict the goal state of object rearrangement. We also present an LLM-assisted trajectory generator to determine the order of rearrangement actions and to generate the corresponding motion trajectories for the robot. We experimentally demonstrate that DiffLOR outperforms all baseline methods in both simulated and real-world scenarios containing unseen objects
